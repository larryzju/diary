motion

C-c C-n Next heading
C-c C-p Previous heading
C-c C-f Next heading same level
C-c C-b Previous heading same level
C-c C-u Backward to higher level heading

Structure editing

M-RET 插入新的 heading（同级）
M-S-RET 同上，设为 TODO
M-left/right Promote/demote heading
M-S-left/right Promote/demote current subtree
M-S-up/down move subtree up/down
C-c C-w Refile to a difference location
C-x n s/w narrow buffer to current subtree

Sparse trees

折叠全部内容，除了匹配的章节

例如可以用 TODO, tags, properties 过滤

C-c / select a sparse-tree
C-c / r match is also highlighted

Plain Lists

即 items，用过提供额外的层级表示

  • Unordered list: -, +, *
  • Ordered list: 1., 1)
  • Description list: ::, 分割 description 与 term

缩进很重要（类似于 python），不一致的缩进或者两个空行表示结束

TAB
折叠
M-RET
插入同级别， C-u M-RET 插入 heading
M-S-RET
插入 checkbox
M-S-up/down
上下移动
M-left/right
减少/增大缩进
M-S-left/right
同上，包括 subitems
C-c C-c
切换 checkbox 状态
C-c -
切换不同的 bullets 标识

Tables

表以 "|" 开头，如下所示 

|Name |Phone|Age| 
|-------+-------+-----|
|Peter| 1234| 17|
|Anna | 4321| 25|
  • 通过 TAB 或 RET 或 C-c C-c 来对齐
  • ‘|-’ 开头表示这是一条水平线,或者 C-c RET

常用快捷键有：

  • Creation and convertion
    C-c |
    convert the active region to table
  • Re-aligning the field montion
    C-c C-c
    Re-align the table
    TAB
    Re-align, and move to the next field
    S-TAB
    Re-align, and move to previous field
    RET
    Re-align, and move down the next row
  • Column and row editing
    M-left/Right
    向左向右移动一列
    M-S-left
    删除当前列
    M-S-right
    在左边插入一列
    M-up/down
    向上向下一行
    M-S-up
    删除当前行
    M-S-down
    插入一个新行
    C-c -
    插入水平线
    C-c RET
    同上，并移动光标向下一行
    C-c ^
    排序
