Here's a short cheatsheet for Markdown language. Mainly from https://guides.github.com/features/mastering-markdown/ and some contents are from https://daringfireball.net/projects/markdown/syntax#header.

Advanced of pure text markup language

  1. collaborative and version control
  2. any editor
  3. unique format
  4. focues on content
