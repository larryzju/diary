概述

Kubernete cluster orchestration system，关于如何将容器化的应用（application）部署到集群上的平台。

提供辅助工具，使应用能长期稳定运行，提供快速迭代部署能力。

概念

k8s cluster

  • 由 master 和多个 node 组成，对外提供一个统一的接口视图。
  • 类比于分布式的操作系统，负责对提交的应用进行发布（到某个 node）和调度
  • master 负责协调 node 运行，维护状态，提供扩展能力、更新能力
  • node 上运行有 kubelet 和具体的容器化工具（docker 或 rkt），受 master 指挥，完成工作
